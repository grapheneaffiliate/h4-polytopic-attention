## Mechanic
Sokoban-style crate pushing. Player moves on 64Ă—64 grid, pushes crates to target positions. Step budget per level.

## Win Condition
`ymzfopzgbq()`: ALL target sprites have matching crates AND all crates not in exclusion set `zmqreragji`.

## Actions
- ACTION1: move up (dy = -4)
- ACTION2: move down (dy = +4)
- ACTION3: move left (dx = -4)
- ACTION4: move right (dx = +4)
- ACTION5: pick up / place box (grab adjacent box, or drop carried box)

## Carry Mechanic (nsevyuople)
- `nsevyuople`: dict mapping carrier -> carried sprite
- `zmqreragji`: reverse mapping (carried -> carrier)
- ACTION5 near box: grabs it (`xpcvspllwr`)
- ACTION5 while carrying: drops it (`kqrtstlzkg`)
- When moving while carrying: both player AND carried box move together
- Movement blocked if either player OR carried box would collide
