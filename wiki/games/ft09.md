# FT09
**Status:** 6/6 DONE | **Tags:** click | **Baselines:** 17, 19, 15, 21, 65, 26

## Mechanic
Lights-Out constraint puzzle. Click cells to cycle colors (mod p). Constraints specify which cells must equal/not-equal target colors.

## Solver: `scripts/solve_ft09.py`
GF(p) Gaussian elimination. For 2-color levels: GF(2) with null space search for NOT_EQUAL constraints. For 3-color (level 3): enumerate 2^16 NOT_EQUAL assignments, solve each with GF(3).

## Key Discoveries
- Level 4: NTi tiles have custom influence patterns (clicking affects neighbors, not just self). Pixel-6 at position (0,1) means clicking cycles tile AND tile above.
- Level 5: ZkU tiles with tags=["NTi","gOi"] — same NTi influence as level 4. Coordinates from other session needed adjustment.
- Clicks commute: order doesn't matter, only total click count per cell.
